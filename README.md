# 🎬 Movie Search App

A Vue 3 movie search application built with the Composition API and powered by the [OMDb API](https://www.omdbapi.com/).

## Features

- **Instant search** by movie title
- **Movie cards** with poster, title, year, and type badge
- **Responsive grid** — 4-5 columns on desktop, 2-3 on tablet, 1-2 on mobile
- **Loading spinner** with film-reel animation
- **Error handling** for failed requests and empty results
- Dark cinematic UI with smooth animations

## Getting Started

### 1. Clone and install

```bash
cd movie-search-app
npm install
```
